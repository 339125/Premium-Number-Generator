# ETH靓号生成器

一个基于Auto.js的以太坊靓号钱包生成器，支持自动检测余额并保存有余额的钱包私钥。

## 功能特点

- **靓号规则设置**：支持自定义前缀和后缀匹配
- **自动余额检测**：通过ETH主网RPC自动检测钱包余额
- **私钥安全保存**：发现有余额的钱包自动保存到本地
- **实时监控**：显示生成数量、检查数量、发现数量
- **多线程生成**：支持并发生成提高效率

## 使用方法

1. 将 `eth_vanity_generator.js` 文件复制到安卓手机
2. 打开Auto.js应用
3. 导入并运行脚本
4. 设置靓号规则后点击"开始"按钮

## 文件说明

- `eth_vanity_generator.js` - 主脚本文件
- `SPEC.md` - 项目规范文档

## 注意事项

- 生成靓号是概率性事件，建议设置较短的前缀/后缀（2-4位）
- 钱包会保存在 `/sdcard/ETH_Wallets.txt`
- 请勿泄露私钥文件
